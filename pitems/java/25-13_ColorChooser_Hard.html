Problem Description

This program creates a windowed program using Java Swing components that displays a colored rectangle. The color can be changed by both moving the RGB sliders or editing their text fields.

User Documentation

The user can manipulate the color being displayed by the rectangle by either moving one of the RGB sliders, which also updates the corresponding text fields, or editing one of the text fields, which also updates the corresponding RGB slider.

Developer Documentation

The ColorChooser class extends JFrame, and creates a GUI to choose a color for the displayed rectangle from the ColorPanel class by using JSliders and JTextFields to enter data.

The ColorPanel class extends the JPanel class, and overrides the paintComponent method to draw a rectangle based on the size of the panel and the values of its Color variable. It also provides public setRed, setGreen, and setBlue methods to update the corresponding RGB value and call the repaint method.

ChangeEvents and ActionEvents are generated from the sliders and text fields and used to call the ColorPanels methods and update the GUI.
