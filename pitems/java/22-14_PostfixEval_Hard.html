Problem Description

This program takes in a String in postfix notation, and provides functionality to evaluate it.

User Documentation

The user can add to the postfix string by entering an integer or operator in the text field and pressing enter or pressing the "Add to Postfix String" button. The current postfix string will is shown in the "Current String" box. The string can be evaluated by pressing the Evaluate button or cleared with the Clear button.

Developer Documentation

A valid postfix string is assumed.

The PostfixEvaluatorGUI class creates the window and implements an ActionListener that is configured to listen for events from the Add to String, Evaluate, and Clear buttons as well as events from the top text field. The class also initializes a PostfixEvaluator object to store the postfix StringBuffer object and perform the evaluation.

The PostfixEvaluator performs the postfix evaluation by reading the StringBuffer from left to right. It pushes numbers to an integer stack, and when an operator is encountered, the top two values are popped from the stack and evaluated using that operator. The result is pushed back on the stack, and this continues until the end of the string is reached. The final value popped from the stack is the final result to be displayed in the Evaluation field.

The string is traversed character by character using spaces to separate both integers and operators from each other.
