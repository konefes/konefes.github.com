16-22_MorseCode_Medium

Problem Description

This program creates a window that provides the functionality to convert plain text into Morse code. Additionally, Morse code can be converted into plain text.

User Documentation

The user can enter plain text in the left text field to be converted to Morse code, or Morse code in the right text field to be converted to plain text.

Developer Documentation

The code consists of the MorseCodeWindow class that extends JFrame to create the user interface, and an inner class ButtonHandler that handles events from button presses to convert one way or the other.

Upon pressing "Text to Morse", the string from the text area is split by spaces to form words. The words are looped through letter by letter, and the Morse equivalent of each character is stored in a string followed by a space. At the end of a word, three spaces are inserted. This continues until the entire message is encoded.

Upon pressing "Morse to Text", the string from the Morse area is split by sequences of 3 spaces to form words. The words are looped through, and each Morse representation of a character is separated by a single space. The text equivalent of each Morse sequence is stored in a string. At the end of a word, a space is inserted. This continues until the entire message is decoded.

Two arrays store corresponding entries for the text characters and their equivalent Morse string to allow for easy encoding/decoding.
