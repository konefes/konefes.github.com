8-16_HugeInt_Medium

Problem Description

This program creates HugeInteger objects that contain an array representing up to 40 digits of an integer. Compare and arithmetic functions are also provided.

User Documentation

Upon starting, two random HugeIntegers h1 and h2 are created. Operations are performed to compare to zero and each other. After that, the two HugeIntegers are added and subtracted from each other. All results are printed to the display.

Developer Documentation

The HugeInteger class contains an integer array of size 40. It can be filled by the parse method that takes a string and converts it to the digits in the array. Array entries can also be set individually.

The HugeIntegerTest class uses Math.random to create two HugeInteger objects with a random number of digits between 1 and 40. It also fills each digit with a random number between 1 and 10. It then tests the isZero, isEqualTo, isLessThan, isGreaterThan, add, and subtract methods and prints the results.
