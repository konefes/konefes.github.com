19-10_Quicksort_Medium

Problem Description

This program creates window with options for the user to add integers to an array. The integer array can be sorted at any time using the Quicksort algorithm.

User Documentation

Integers can be entered in the text field at the top, and are shown in the main text area. The user can enter as many integers to the array as needed. The Quicksort button sorts the array and the clear button resets the array to 0 elements.

Developer Documentation

The code consists of the QuickSortGui class that extends JFrame to create a user interface. QuickSortGui also implements ActionListener to provide event handling capabilities for the "Add Integer to Array", "QuickSort", and "Clear Array" buttons. Integers can be added from the top text field, and the Current Array field is updated whenever a new integer is added to the array. The Sorted Array field is updated upon sorting.

The Quicksort algorithm works using a partitioning step that takes the first element of the unsorted array and determines its final location in the sorted array (everything to the left is smaller, everything to the right is larger). The subarrays of smaller and larger elements are sorted recursively using the same method. When a subarray is down to one element, that element is in its final location.
